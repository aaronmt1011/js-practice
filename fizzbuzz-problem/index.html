This project deals with FizzBuzz which is a children's game that teaches about division.

A number (n) is picked from 1 - 100.

This program prints out numbers from 1 to picked n (number).

However for certain numbers, it will print fizz, buzz, or fizzbuzz instead.

  • For numbers that have multiples of three (ex. 3, 6, ...), it will print Fizz instead.
  • For numbers that have multiples of five (ex. 5, 10, ...), it will print Buzz instead.
  • For numbers that have multiples of both three and five (ex. 15, 30, ...), will print FizzBuzz.
